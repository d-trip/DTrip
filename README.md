Golos.mapala client

> Nuxt.js

Env:
  API_QL_URL
 
 
## Run local
```API_QL_URL=https://golos-ql.mapala.net/ yarn run dev```

## Run docker
* Create .env file.

* ```docker-compose up -d --build```

For detailed explanation on how things work, checkout the [Nuxt.js docs](https://github.com/nuxt/nuxt.js).
